Dashboard · Aug 2022

Python, Streamlit, Google API

Overview

A simple dashboard built from faculty-shared CSV data that helped 400+ freshman students at the Faculty of Computer Science, Universitas Indonesia, find classmates in their first term. Students could search by name or student ID to see which classes they shared with others, making it easier to connect during orientation.

The first iteration was built and released on the same day the faculty published the class schedule sheet, allowing students to immediately find their classmates before the term started.

How It Works

  • Faculty-provided CSV files containing student class schedules are parsed and indexed.
  • A Streamlit dashboard lets students search by name or student ID.
  • The app displays shared classes between any two students, helping freshmen find familiar faces in their schedule.
  • Google API collects user feedback and reviews from students.
